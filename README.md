# Virtual Classroom
This is an Application which presents classroom with students

## Technologies :
* Backend: Java, Spring(Boot, Data, Security), WebSocket
* Frontend: React, HTML, CSS, Bootstrap
* Common: Git, Maven, Rest API, JWT
* Database: H2
